# Projeto Amigo Secreto

Este é um projeto que utilizei para praticar **JavaScript** e **manipulação do DOM**.

A estrutura em **HTML e CSS** já estava pronta, e minha parte foi desenvolver toda a lógica da aplicação utilizando **JavaScript**.

O objetivo do projeto é criar um **sistema de sorteio de amigo secreto**, permitindo adicionar participantes, exibir a lista de nomes, realizar o sorteio automaticamente e mostrar quem tirou quem.

---

## Funcionalidades Implementadas

- Adição de nomes na lista de participantes com validação para evitar nomes vazios
- Verificação para impedir a inserção de nomes duplicados, independentemente de maiúsculas ou minúsculas
- Exibição dinâmica dos nomes separados por vírgulas
- Possibilidade de remover um nome da lista ao clicar nele
- Embaralhamento aleatório dos participantes para garantir sorteio justo
- Exibição do resultado do sorteio mostrando as correspondências entre participantes
- Função para reiniciar o sistema, limpando a lista e os resultados

---

## Tecnologias Utilizadas

- **HTML** (estrutura da aplicação)
- **CSS** (estilização da interface)
- **JavaScript** (lógica e manipulação do DOM)

---

## Como Funciona

1. O usuário digita o nome de um participante.
2. Ao clicar em **Adicionar**, o nome é incluído na lista, se válido e não duplicado.
3. A lista de participantes é exibida na tela separada por vírgulas.
4. Ao clicar em **Sortear**, o sistema embaralha os nomes e realiza o sorteio.
5. O resultado do sorteio é exibido, mostrando quem tirou quem.
6. O usuário pode clicar em qualquer nome na lista para removê-lo.
7. A opção **Reiniciar** limpa a lista de participantes e os resultados do sorteio.
